Java Drawing Project

This was a highschool project the goal of which was to demonstrate the fundementals of calculus through a visual program.

Skills:

Java, J-Frame

Outcome:

For this project I created a curve class from scratch by partitioning curves from lines in J-frame. This was done to illustrate the fundimental principle of integrals. I then created a treadmill of curves which move across the window with a marker following them. When each curve reaches the end of the screen it is moved to the back again and its attributes are randomized to give the illusion of an infinite random series of curves.
