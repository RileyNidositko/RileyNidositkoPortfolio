Terrier Motorsport Cooling Re-design

This project was for the Boston University Terrier Motorsport team, which is BU's Formula Hybrid+Electric SAE team. The goal of this project was to ensure optimal air flow to the cooling system by designing a re-mount and ducting for the cooling system radiator. This project is still in progress.

Skills:

Fusion CAD, Mechanical Design, Project Planning, Welding, Composite Manufacturing, Machining

Important Requirements:

All cooling components must stay behind the firewall, System must be exhibit a 1.2 FOS for heat out vs heat in, Must not interfere with other systems (tractive lights, GLV box, etc.) System must must allow for 2.016 m^3/s of air to the radiator as per radiator specs.

Outcome:

This project is still in progress. The analysis and designs are finalized so I am now working through the manufacturing step. The design consists of a main duct along with ramps to focus the air only into the air recieving part of the radiator. These ramps have already been manufactured so the last step is to make the main duct. This will be done with a fiberglass layup onto a custom bent sheet metal mold.

I am helping create a new procedure for our composite layups for all of our aerodynamic pieces so I have been testing our techniques with other Terrier Motorsport members for verification. Part of our new layup procedure involves a new tool called a finned composite roller. Terrier Motorsport did not have any of these tools already and I couldn't wait to order one so I made one myself (see images below). After the new procedure is verified and the problems are fixed we can move on to the final layup.

So far through this project I have learned CAD design on Fusion and practiced welding and machining with a waterjet. I have also learned significantly about composite manufacturing techniques.
