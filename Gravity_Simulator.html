Gravity Simulator

This was a project I did in highschool for a javascript coding course. The goal of this program was to simulate the motion of planets under gravity.

Skills:

Javascript

Outcome:

For this project I was unable to make real classes (due to constraints with the course) so instead I simulated classes by organizing planet objects using an index number across a number of arrays. Because of this approach I was able to continously add new planets by appending new variables to the arrays with no limit other than the proccessing power of the system.
